
----------------------- TAS - Telegram Accs Saver -----------------------

Сохраняет аккаунты телеги, и когда будет надобность зайти, даст код :)

Перед заходом в акк, запустите скрипт и выберите 3-й модуль, чтобы получить код, для авторизации в телеге
Иначе код, вероятнее всего, пройдёт мимо

Чтобы получать коды от новодобавленных аккаунтов, нужно перезапустить скрипт!

Для работы, нужно залезть в файл config.py и заполнить пустоту :)
